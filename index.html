🎬 Storyboard director

Turn this on to show a 🎬 Generate Frame button by the message bar. Tap it whenever you want a storyboard frame from the latest story moment — nothing generates automatically, so you only spend when you choose to. Each frame also drafts a Wan 2.2 video prompt you can copy. Returning characters reuse their saved looks from your cast.

Storyboard mode

👥 Cast — this story

Character sheets for the current game. You manage these manually — add a character, tap ✨ to write a physical description, and set a reference image (upload, generate, or use your last frame) to lock that character's look across storyboard frames. The director reuses these sheets to keep returning characters consistent.

📖 Story Setup

Prime a new game. Premise & tone steer the chat; visual style is applied to every storyboard frame. Manage characters in the 👥 Cast menu. Per-story — reopen and tweak anytime. Tap ✨ on any field to have the AI write or expand it from a rough idea.

💾 Save / Load Game

Save the current game as a code — story, cast, Story Setup, and storyboard settings. Images/videos are not included (save those separately); frames regenerate as you continue. Paste a code to recreate a game in a new thread.
