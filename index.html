🎬 Storyboard director

While you roleplay, this auto-builds an inline storyboard — generating a frame whenever a new character, location, or wardrobe change appears (or when you type "giggity"). Each frame also drafts a Wan 2.2 video prompt you can copy.

Storyboard mode
Auto-generate frames as you play

👥 Cast — this story

Character sheets for the current game. New characters are added automatically as the story introduces them; the storyboard director reuses these to keep returning characters looking consistent. Edit any time — your edits stick.

📖 Story Setup

Prime a new game. Premise & tone steer the chat; visual style is applied to every storyboard frame; starting characters seed your cast. Per-story — reopen and tweak anytime.

💾 Save / Load Game

Save the current game as a code — story, cast, Story Setup, and storyboard settings. Images/videos are not included (save those separately); frames regenerate as you continue. Paste a code to recreate a game in a new thread.
